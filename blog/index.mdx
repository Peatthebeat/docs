## Writing Tips

- **Document as you go**: Write posts while working on projects
- **Share challenges**: Others learn more from problems than successes
- **Include code**: Show actual examples and configurations
- **Link to docs**: Reference your main documentation for context

## Getting Started

Create a new blog post:

```bash
# Create a new dated post
touch blog/2024-11-14-my-first-post.mdx
```

Add frontmatter:

```mdx
---
title: "Your Post Title"
description: "Brief description"
date: "2024-11-14"
---

Your content here...
```

Then add it to the Blog section in `docs.json`.
